## Task 1:

```bash
Create an interactive web app with a button that generates a random fantasy character when clicked. 
Each character should have a random Name and a Class (like Warrior, Mage, Rogue). 
Display the character's name and class in the app when the button is clicked.
```
## Task 2:

```bash
Add a feature to generate and display a cartoon or video-game-style portrait for the generated character. 
Include two buttons: "Generate Portrait" and "Regenerate Portrait". 
Each portrait should match the character's class.
```
## Task 3:

```bash
Refine the user interface with a fantasy theme.
Give the app a dark, textured background that looks like an ancient alchemist's workbench.
Use a fantasy-style font for the character's name, such as "Uncial Antiqua" or "Cinzel Decorative".
Keep the layout clean and readable.
Add a glowing border or subtle magical particle effect around the main content.
```
## Task 4:

```bash
Add a "Generate Backstory" button. When clicked, create a unique, one-to-two-sentence origin story for the current character and display it.
```
## Task 5:

```bash
Reformat the UI as a bordered player card. Add random stats: Health, Mana, and Strength. Include a "Save to Deck" button that saves the current character to a 'My Deck' list in the app.
```
